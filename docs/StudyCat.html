Introduction

Background

此專案在 2020 Line Fresh 黑客松所做，我們是一組五人團隊～在比賽最後我們抱走了最佳評審獎！

Problem

發現大學生在規劃待辦事項上的執行成效低落、事務繁雜眾多；以及生活規劃會高度受同儕影響

Solution

經過調查、分析、及設計思考，我們決定以 Line Chatbot 作為服務平台，提供蕃茄鐘、任務規劃平台、等等功能

Tools

Figma, Procreate, Adobe Illustrator, WebStorm

My Role

UIUX Designer/Front-end Developer

User Survey

從93份有效問卷中，發現超過六成的人並不會規劃待辦事項的優先順序並安排執行時間因此我們設計出解決這個問題的產品！

Define Goal

經過發想和設計思考，我們以學生為目標族群，打造了客製化的 Line 聊天機器人服務來增進學習效率

Product Features
