UI Optimization

帳號註冊信箱驗證流程優化

帳號註冊的信箱驗證在優化前需要使用者手動將寄到信箱中的驗證碼複製貼上，除了安全性問題外，流程上不太順暢，需要先填寫信箱帳號並驗證完之後，才可以將註冊資料上傳。

  • 介面斷點一：透過驗證碼驗證信箱，需要從信箱複製後貼上到註冊頁面，會有安全性及易用性問題。

    改善方式：將信箱的驗證碼改完驗證按鈕。

  • 介面斷點二：流程上需要將信箱完成註冊後才可以將其他註冊資料上傳，造成體驗不好導致轉換率低。

    改善方式：在驗證流程上先將資料送出，後續有空再進行信箱驗證。

在解決斷點時，我利用競品分析調查其他平台的信箱驗證流程後，先畫出 Flow Chart 將流程可視化方便與團隊夥伴討論，後續再製作 Mockup，並且使用 HTML 大致切出優化後的驗證信件模板，以確保設計和開發上溝通的效率。
