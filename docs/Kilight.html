Kilight

Introduction

Background

此專案是為參加2020林口跨界超越競賽的產品
最終晉級到複賽，比賽結束後，我們照著評審給的評語，將專案迭代、優化

Challenge

讓正經歷子女離巢期的中年婦女，能有一個管道重新聯繫親子間的關係、欲解決中年婦女想與離家子女無法保持聯繫的問題。

Idea

以運動任務及交流信箱為親子共同任務，可獲得實質蔬果作為獎勵之養成類遊戲應用程式

Outcome

滿意度（7.2/10）

Tools

Figma、Miro、Procreate

Role

在團隊中我的角色是
Team Leader/UX Researcher/UI Designer

Define User

原始動機

我們的起心動念是發現在市面上沒有以中年婦女為主要目標族群
的運動（健康照護）產品，因此想發展出以下產品讓這個族群可以被關心與照顧。

初步訪談

初步訪談後，發現在這群婦女中大部分人因為身體狀況所以都有運動需求或習慣、
也同時因經歷小孩離巢期階段，所以大部分人希望多一點親人的陪伴，
所以我們決定以這兩項初衷，去發展出一個增進親子關係的運動軟體。

Persona

主要TA:高運動動機媽媽/次要TA:離巢期孩子

User Research

研究問題

痛點

世代觀念價值觀相異
和離巢期子女互動機會少
更年期症狀的出現

POV

一位以運動習慣來舒緩更年期症狀且子女正值離巢期的中年婦女，她需要和子女一起活動身體的機會，因為對她來說，能與子女互相關心生活與健康是很重要的

洞見：與子女互相關心生活與健康

HMW

我們如何讓父母與子女間重新找回生活上的連結
我們如何讓親子間互相聯繫變成一件好玩的事情
我們如何提升親子之間相處的儀式感
我們如何利用運動來聯繫親子關係

Usage scenarios

Storyboard

Features

功能架構

以上是Kilight的功能架構圖，接下來會針對交流信箱、運動任務、親子樹三大功能更詳細的說明


交流信箱

以上是交流信箱的架構及頁面，
此功能的發想來源為：親子間大多具有溝通上的問題、婦女期待與小孩像朋友般互動


運動任務

以上是運動任務的架構及頁面，
此功能的發想來源為：小孩與婦女皆需運動，因此發想以運動創造連結產生共感、中年婦女較習慣從事低強度運動並以運動舒緩更年期症狀


親子數

以上是親子樹的架構及頁面，
此功能的發想來源為：單純維繫家庭關係之產品的取代性高且黏著性較低、農場養成遊戲化系統、互動視覺化

Usability Test

易用性及滿意度



易用性訪談分析

以三組符合目標組群的親子做易用性測試，
以了解易用性、滿意度、及確認使用流程。

Result

使用者專業人士回饋

使用者：「以養成方式督促運動與親子關係，有趣。產品功能之間能夠互相搭配，玩遊戲的時候能夠有所真實收穫，大大降低使用APP的罪惡感」
發展心理學教授：「可以幫助「不住在一起的親子」增進其親密關係以及連結。如果是自己喜歡的運動有在軟體裡面，會大大增加使用頻率。」


未來發展

我們將會以易用性收到的結果來改良產品的功能，
並將經過更多的調查及訪問來迭代出更符合使用者的產品，
期待這項產品能讓這個世界更好！
