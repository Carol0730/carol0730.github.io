Introduction

Background

此專案是在2020 Line Fresh 黑客松所做，
在比賽最後我們抱走了最佳評審獎！
This project was made during the 'Line Fresh 2020' hackathon.
In the end, we won 'The Best Judging award'!

Problem

我們發現學生在規劃待辦事項上的執行效果成效低落
We found that students' implementation for scheduling is ineffective.

Result

經過調查和分析，我們創在了 Line 聊天機器人
裡面包含蕃茄鐘、任務規劃平台、等等
After investigation and analysis, we created a ChatBot
including Pomodoro, a task-managed platform, and so on.

Role

我在此專案的角色是設計介面及部分前端工程師
UIUX Designer/Front-end Developer

User Survey

從93份有效問卷中，超過六成的人並不會規劃待辦事項的優先順序並安排執行時間
因此我們設計出解決這個問題的產品！
From the 93 valid questionnaires, it can be seen that more than 60% of students will not plan the priority order of the to-do items and arrange the execution time. Hence, we designed and created the product based on this situation!

Define Goal

經過發想和設計思考，我們以學生為目標族群，
打造了客製化的 Line 聊天機器人服務來增進學習效率
After design thinking, we chose students as target audience and create a customized Line ChatBot service in order to improve learning efficiency.

Product Features

在服務中，有以下五個主要功能
In this ChatBot, users can use the following five functions.


此兩項功能為記錄待辦事項並瀏覽未完成的任務
These two functions are to create a task list and display the currently scheduled tasks.


蕃茄鐘可以計算專注時間和休息時間，此方法已獲證能有效增強執行者的專注度與效率。
The Pomodoro is to timing the focus and rest time.
In other words, we can use it to count the time we are concentrating or relaxing.


使用者可以使用筆記共享在群組間分享筆記，並使用討論版詢問或回答問題
Users can share notes with group friends on the note sharing,
and post or answer questions on the discussion board.

Technology Architecture

在LINE聊天機器人中使用Line Messaging API, LIFF來串接機器人功能
Using LINE Messaging API, LIFF, users can record and plan to-do list in real-time.


串接的LIFF網頁前後端使用以上技術
The back-end uses Google Sheets as the database,
and the front-end uses Bootstrap, Ant Design, and Pug as the framework
